# Auto_Thin
An end-to-end R pipeline that takes [3DFin] (https://github.com/3DFin/3DFin) tree-detection outputs from Terrestrial Laser Scanning (TLS) and Mobile Laser Scanning (MLS), applies competition-index-based thinning decisions, aligns all data sources into a common geodetic reference and produces maps and statistics against hand-recorded inventory data.
